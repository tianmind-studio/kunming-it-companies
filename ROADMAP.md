# Roadmap

这个项目优先服务 GitHub 搜索和本地技术社区协作，不追求复杂产品形态。

## Phase 1: 可被发现

- [x] 公开 GitHub 仓库
- [x] README 覆盖“昆明 IT 公司 / 昆明互联网公司 / 昆明软件公司”等关键词
- [x] `COMPANIES.md` 生成 GitHub 可搜索索引
- [x] GitHub Pages 提供轻量浏览页面

## Phase 2: 可信数据

- [x] 给每条记录增加公开来源
- [x] 区分 `official_site`、`official_profile`、`community_pending`
- [x] 页面展示已核验和待复核数量
- [ ] 把待复核记录逐步补成官网或官方主页来源
- [ ] 补充公司所在区县、招聘页、技术方向

## Phase 3: 社区参与

- [x] 新增公司、修改公司、复核公司的 issue 模板
- [ ] 整理一批适合新贡献者处理的待复核任务
- [ ] 发布到本地开发者群、V2EX、掘金、知乎等入口
- [ ] 每月做一次数据复核

## Phase 4: 更好用

- [ ] 增加按区县浏览
- [ ] 增加按招聘活跃度或最近核验时间排序
- [ ] 增加 CSV 导出
- [ ] 增加贡献者名单
