# Roadmap

Kunming Tech Radar 会从公司索引逐步扩展为本地技术机会资料库。

## Phase 1: 数据结构

- [x] 建立公司、岗位、活动、社群、政府项目五类 CSV
- [x] 保留早期公司索引，作为迁移和复核基础
- [ ] 为 CSV 增加基础校验脚本
- [ ] 明确数据许可和引用方式

## Phase 2: 可信来源

- [ ] 把待复核公司补充为官网或官方主页来源
- [ ] 建立岗位和活动的过期标记规则
- [ ] 建立政府项目线索的来源格式
- [ ] 每条记录保留来源链接和核验日期

## Phase 3: 机会雷达

- [ ] 建立按区县、行业、机会类型浏览的静态页面
- [ ] 增加 CSV 到网页的自动生成流程
- [ ] 支持按更新时间查看新增线索
- [ ] 增加贡献者名单和更新日志

## Phase 4: 社群与线下

- [ ] 建立昆明技术机会雷达社群
- [ ] 试运行本地技术活动和创业资源整理
- [ ] 评估是否开放付费内圈、线下活动或整理版资源库
