# Contributing

感谢你补充昆明 IT 公司信息。

## 添加公司

优先使用 GitHub issue 模板。直接 PR 也可以，请修改 `data/companies.json` 并运行：

```bash
npm run validate
```

## 必须提供来源

每条记录至少需要一个公开来源：

- 官网
- 官方公众号文章
- 官方招聘主页
- 工商公开信息页
- 可信媒体报道
- 公司官方 GitHub / 开源主页

## 文字要求

请用中性表述：

- 可以写：`官网展示其提供医疗信息化产品。`
- 不建议写：`昆明最强医疗信息化公司。`
- 不写未经核实的加班、薪资、争议、个人联系方式。

## 数据复核

如果你发现公司官网打不开、公司已搬迁、业务方向变了，欢迎提交 `Update company` issue 或 PR。
