# Kunming Tech Radar

Kunming Tech Radar is a public, source-aware dataset for technology opportunities in Kunming and Yunnan: companies, jobs, events, communities, and public digital project leads.

The primary README is maintained in Chinese: [README.md](README.md).

This project does not rank companies, guarantee jobs, or claim access to internal opportunities. It collects public information and keeps source links whenever possible.

Recent additions:

- `data/source-leads.csv` keeps at least five public source leads for each major direction.
- `data/communities.csv`, `data/events.csv`, and `data/gov-projects.csv` now contain conservative public source entries.
- A small local community pilot has started; the repository records it as a public-facing community entry without publishing private group QR codes.
- Recruiting-platform search pages are treated as discovery leads only; they do not imply active hiring.
