# Community Playbook

这个项目的增长不靠单独网站流量，靠 GitHub 搜索、社区转发和持续 PR。

## 对外介绍

短版：

```text
我整理了一份昆明 IT 公司 / 昆明互联网公司开放清单，放在 GitHub 上，方便本地开发者、求职者和外地团队了解昆明技术生态。欢迎补公司、补官网、纠错。
https://github.com/tianmind-studio/kunming-it-companies
```

稍长版：

```text
昆明技术公司信息比较分散，所以我做了一个开源清单：昆明 IT 公司 / 昆明互联网公司地图。它不是排名，也不是黑名单，只收录有公开来源的信息。现在有一批公司还处于待复核状态，欢迎本地朋友帮忙补官网、招聘页、业务方向或纠错。
```

## 适合发布的地方

- 昆明本地技术群
- V2EX 城市 / 职场 / 程序员节点
- 掘金沸点
- 知乎问题：昆明有哪些互联网公司、昆明程序员发展、云南软件公司
- GitHub 个人主页或置顶仓库

## 邀请贡献时怎么说

重点不是“求 star”，而是降低参与门槛：

- 发现少了公司：开 `Add company` issue
- 发现信息错了：开 `Update company` issue
- 找到了官网：开 `Verify company` issue

## 质量底线

- 不写未经核实的负面评价
- 不写私人手机号、私人微信
- 不做排名
- 不把待核验信息说成确定事实
