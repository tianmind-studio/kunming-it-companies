# Community Guidelines / 社区协作边界

昆明技术机会雷达 / Kunming Tech Radar 希望成为一个可信、克制、可复核的本地技术机会地图。

## 协作原则

- 用公开来源说话。
- 标注不确定性，不要假装知道。
- 先修正事实，再讨论判断。
- 尊重公司、学校、社群和个人隐私。
- 对新贡献者友好，帮助他们把线索补成可复核记录。

## 不接受的内容

- 私人联系方式、群二维码、聊天截图。
- 无法公开核验的薪资、裁员、加班、纠纷、负面爆料。
- “内部机会”“包过”“保证推荐”等不可验证承诺。
- 对公司、学校、个人的攻击性描述。
- 为了数量而编造公司、岗位、项目或活动。

## 讨论方式

- 新线索优先开 Issue。
- 数据修改优先发 PR。
- 不确定记录先标 `community_pending`。
- 有争议时保留来源和中性说明，避免主观判断。
