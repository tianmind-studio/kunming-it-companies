## How to use this plan

1. Open the current source first.
2. Look for stronger public sources only: 官网、企查查/天眼查公开页、政府项目页、公众号文章、招聘页、公开活动页。
3. If a stronger source is found, update `data/companies.json`, keep the old source in `source_urls` when useful, and update `last_checked`.
4. If no stronger source is found, keep the record conservative: `community_pending`, low confidence, empty district, or `unknown` opportunity hints.
5. Run `npm run generate:companies`, `npm run export:csv`, `npm run generate:data-quality`, and `npm run validate`.
