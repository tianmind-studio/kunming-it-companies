# Opportunity Radar / 机会雷达说明

机会雷达关注“本地技术机会”而不只是公司名单。

## 机会类型

| 类型 | 说明 |
| --- | --- |
| 公司机会 | 本地技术公司、外地公司昆明团队、传统企业数字化团队 |
| 招聘机会 | 实习、校招、社招、远程、兼职、项目制合作 |
| 活动机会 | 技术分享、创业活动、黑客松、校园活动 |
| 社群机会 | 开发者群、高校社团、创业社群、行业协会 |
| 政府项目 | 政府采购、智慧城市、政务系统、数据平台、产业数字化项目 |

## 如何判断一条线索是否值得收录

优先考虑三点：

1. 是否和昆明或云南有关。
2. 是否和技术、软件、AI、数字化、数据、信息化有关。
3. 是否有公开来源可以核验。

如果一条线索只有聊天截图或口头转述，暂时不建议直接写入数据集。可以先开 Issue 讨论，并等待更可靠来源。

## 当前新增的数据入口

- `data/source-leads.csv`：9 个方向 × 至少 5 条公开来源种子，主要用于继续发现候选公司和招聘主页。
- `data/communities.csv`：高校、科技门户、活动平台、本项目试运行交流群等公开社群/组织入口。
- `data/events.csv`：技术、AI、数据、创业活动的公开搜索入口或活动详情候选。
- `data/gov-projects.csv`：公共资源交易、政府采购、科技项目等政府数字化项目入口。

这些入口不直接生成“机会已存在”的结论。只有具体公司、具体活动或具体项目能被公开页面核验后，才应进入对应主数据字段。

## 更新节奏

目前项目还在早期阶段，先以手动维护为主。后续可以逐步加入半自动信息源，但每条记录仍需要保留来源链接和核验日期。
