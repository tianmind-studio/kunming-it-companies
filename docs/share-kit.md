## 朋友圈文案

我做了一个小项目：昆明技术机会雷达。

它不是招聘中介，也不是公司排名，而是一个开源的本地技术机会数据库。

现在已经收录了昆明/云南技术相关公司和机构线索，并把公开来源、核验状态、适合人群、方向标签都结构化了。除了公司索引，也开始整理公开招聘入口、技术活动、社群和政府数字化项目来源。

更重要的是，已经有人通过线上入口找到我，我也开始试运行一个小群：后续会把群里的补充线索继续沉淀回开源数据里，而不是只停留在聊天里。

适合：

1. 昆明/云南学生找实习、找项目方向
2. 开发者了解本地技术团队
3. 自由职业者找外包和合作线索
4. 创业者观察本地数字化机会
5. 企业/机构补充自己的公开入口

欢迎补充官网、招聘页、活动页、社群入口。只收公开来源，不收私人联系方式，不做就业担保。

项目地址：
https://github.com/tianmind-studio/kunming-it-companies

目前项目已经出现第一个现实反馈：有人通过线上入口找到维护者，并开始试运行小群。后续重点不是把聊天变成新的信息孤岛，而是把群里出现的公开来源、公司官网、招聘主页、活动页继续沉淀回这个仓库。

## 小红书文案

标题备选：

- 昆明也有技术机会，只是以前太分散了
- 我整理了一张昆明技术机会地图
- 给云南学生和开发者的本地技术公司雷达

正文：

很多人觉得昆明没有技术机会，但真实情况可能是：机会太分散了。

有些在公司官网，有些在招聘平台，有些在政府公告，有些在高校和园区活动里，还有很多藏在公众号和 PDF 里。

所以我做了一个开源项目：昆明技术机会雷达。

它会把昆明/云南技术相关公司、公开招聘入口、活动、社群、政府数字化项目线索，整理成可搜索、可复核、可贡献的数据。

目前更适合这些人：

- 云南学生：找实习方向、毕业设计方向、项目选题
- 本地开发者：了解昆明有哪些技术团队
- 自由职业者：找外包、系统开发、数字化服务线索
- 创业者：观察农业、文旅、医疗、政企、AI 等行业机会

它不做公司排名，也不保证招聘，只保留公开来源。

如果你知道昆明靠谱的技术公司、社群、讲座、招聘主页，欢迎一起补。

## X / Twitter 文案

I started Kunming Tech Radar — an open-source local tech opportunity map for Kunming / Yunnan.

It is not a job board or a company ranking.
It is a source-backed database of local tech companies, public recruiting pages, communities, events, and digital transformation leads.

Why:
- local opportunities are scattered across websites, PDFs, procurement pages, recruiting platforms, and WeChat/college networks
- students need a better entry point than random job-app scrolling
- developers and freelancers need a searchable local map
- founders need weak signals from real industries: agriculture, tourism, healthcare, gov-tech, AI/data, cybersecurity

Repo:
https://github.com/tianmind-studio/kunming-it-companies

Contributions welcome: official websites, public recruiting pages, event pages, communities, and source corrections.

## GitHub Release / Discussion 文案

Title:

Kunming Tech Radar: source-backed local tech opportunity map

Body:

This update turns the repository from a static company list into a more maintainable local tech opportunity database.

What changed:

- Company data uses `data/companies.json` as the primary source.
- `COMPANIES.md` and `data/companies.csv` are generated from JSON.
- Added GitHub Pages static search UI.
- Added contribution templates for company additions, updates, and outdated info reports.
- Added data standards for source type, verification status, confidence score, and privacy boundaries.
- Added `data/source-leads.csv`: 9 directions × at least 5 public source leads.
- Added community, event, and government project source entry files.

This project does not guarantee hiring, ranking, salary, or company quality. It only organizes public, source-backed information and marks uncertainty explicitly.

Good first contributions:

- Add official websites for `community_pending` companies.
- Add public recruiting homepages without inventing job status.
- Add local tech communities, university tech clubs, AI/data/startup events, or park activity pages.
- Report outdated or broken sources.
