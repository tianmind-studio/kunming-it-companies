# 昆明 IT 地图说明

这个文档说明本项目如何理解“昆明及云南技术生态”。

## 覆盖范围

当前项目以昆明为核心，同时保留云南范围内的重要技术线索。收录对象包括：

- 软件开发、AI 应用、SaaS、云服务、数据服务企业
- 系统集成、医疗信息化、农业数字化、文旅数字化、工业数字化团队
- 本地招聘机会、技术活动、创业社群和高校技术组织
- 政府采购、公共资源交易和政策公告中的数字化项目线索

## 不做什么

这个项目不是公司排名，也不是避坑名单。它不评价公司好坏，不发布未经核实的负面信息，也不承诺任何岗位、资源或合作机会。

## 数据状态

记录会尽量区分来源强度：

- `official_site`: 官网或官方页面
- `official_profile`: 官方公众号、招聘主页、官方新闻或可信媒体报道
- `public_list`: 政府名单、公开名录、历史社区清单等二级来源
- `unknown`: 暂时缺少足够来源，需要继续复核

## 更新原则

优先补充来源清晰、对本地技术人有帮助的信息。没有公开来源的信息可以先作为线索，但不应写成确定事实。
