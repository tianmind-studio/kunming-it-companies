# Support / 支持与反馈

For public data corrections, new source leads, outdated links, or questions about the dataset, please use GitHub Issues:

- Add company: <https://github.com/tianmind-studio/kunming-it-companies/issues/new?template=add-company.yml>
- Update company: <https://github.com/tianmind-studio/kunming-it-companies/issues/new?template=update-company.yml>
- Report outdated information: <https://github.com/tianmind-studio/kunming-it-companies/issues/new?template=report-outdated-info.yml>
- Add community or event source: <https://github.com/tianmind-studio/kunming-it-companies/issues/new?template=add-community-or-event.yml>
- Add government project source: <https://github.com/tianmind-studio/kunming-it-companies/issues/new?template=add-gov-project-source.yml>

Do not post private contact details, screenshots, tokens, or personal data in public issues. If a report involves privacy, describe the problem type without reposting the sensitive content.

中文说明：

- 如果你不会提交 PR，直接开 Issue，把公司名、城市/区县、公开来源链接和为什么技术相关写清楚即可。
- 如果你只是看到一个招聘平台搜索页、活动平台搜索页或政府门户首页，请把它当作来源入口，不要写成确定岗位、活动或项目。
- 如果涉及隐私、错误记录、过期链接或不适合公开展示的信息，请使用 outdated/correction 相关模板，并只描述问题类型。
