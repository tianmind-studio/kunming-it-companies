# Support / 支持与反馈

昆明技术机会雷达只维护公开来源支持的数据。它不是招聘中介，不接受无法公开验证的爆料，也不处理私人纠纷。

## 普通用户

不会 GitHub 的用户优先使用国内提交页：

- Submission page: <https://kunming.tianmind.com/submit.html>

你可以提交：

- 公司官网、产品官网、官方招聘页、官方公众号文章。
- 技术活动、园区活动、高校社团、开发者社群的公开页面。
- 政府数字化项目、公共资源交易或采购公告入口。
- 过期链接、重复记录、区县错误、分类错误等更正线索。

请不要提交私人手机号、私人微信、群二维码、聊天截图、无法核实的薪资/裁员/加班/纠纷信息。

## 隐私、侵权或错误信息

如果某条记录可能涉及隐私、侵权、错误展示或不适合公开，请使用 `Data correction / privacy request` 模板。描述问题类型即可，不要在 Issue 里再次贴出敏感内容。

维护处理原则：

1. 先保护隐私和安全边界。
2. 能用公开来源更正的，优先更正。
3. 无法确认但有风险的，先降级为 `community_pending` 或 `outdated`。
4. 明显不符合收录边界的，说明原因后移除。

## 招聘、合作和商业请求

本项目不承诺就业、不承诺外包、不撮合交易，也不为公司做背书。页面中的 `机会提示` 只是检索提示，不能理解为当前正在招聘、正在外包或确定有合作机会。

如果你希望补充公司信息，请提供公开官网、官方招聘页、技术博客、活动页或项目说明，让维护者和社区可以复核。
