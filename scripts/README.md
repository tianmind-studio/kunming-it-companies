# Scripts

这里预留未来的数据采集和校验脚本。

当前阶段以人工整理和人工核验为主，不做自动抓取。后续可以逐步接入半自动信息源，但需要遵守来源网站规则，并保留原始链接和核验日期。

## 可能的信息源

- Boss 直聘、拉勾、脉脉等公开招聘页
- 企查查、天眼查等企业公开资料平台
- Luma、活动行、公众号文章、园区活动页
- GitHub、Gitee、本地开源项目主页
- 政府采购网、公共资源交易平台、政府部门公告
- 高校、园区、协会、创业服务机构公开页面

## 后续脚本方向

- CSV 字段完整性检查
- URL 可访问性检查
- 重复公司/重复岗位检测
- 过期岗位和活动提醒
- 从公开来源生成待人工复核的候选记录

任何自动化脚本都应该服务于复核，而不是绕过复核。不要把未经核验的抓取结果直接当成事实数据。

## 当前脚本

- `validate-csv.mjs`: 检查五类 CSV 的表头和列数。
- `validate-data.mjs`: 校验早期公司 JSON 数据。
- `generate-companies-md.mjs`: 从早期公司 JSON 生成 `COMPANIES.md`。
